## Chaser Effect LED circuit
This is a circuit I created for Resolution Hardware at Hack Club. What this parallel circuit does is that it has two switches and turns off one LED while the other one is off so only one LED is affect. I tried to make an effect where the brightness switches between LED but it didn't really work well for me.
